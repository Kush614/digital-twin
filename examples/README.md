# Demo assets

Files in this folder are pre-loaded sample inputs for the ImpactLens demos. They are committed to the repository so a first-time clone has working demos out of the box.

## `asl-cc-vocabulary.mp4` — Public-domain ASL clip (1.4 MB)

A short ASL vocabulary clip released under public domain by the **Center for Accessible Technology in Sign** and hosted on the Internet Archive. Used as the click-to-load demo on `/asl-demo`.

- **Source:** https://archive.org/details/creativecommonsASL2
- **License:** Public Domain (per the Internet Archive item page)
- **Size:** ~1.4 MB
- **Length:** short (~10 seconds)

If you want a different demo clip (e.g. a cleaner A–Z alphabet), download one yourself with `yt-dlp` (see `app/asl-demo/page.tsx` for the inline tip) and drop it into the page directly — there's no need to add it to this folder.
